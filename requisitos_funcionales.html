REQUISITOS FUNCIONALES

ID Nombre del Requisito Descripción (Técnica) Rol HU Relacionada (Corregida)
RF01 Motor de Búsqueda y Filtrado Básico El sistema debe implementar una búsqueda que permita filtrar actividades por categoría, ciudad o zona, y fecha. Los resultados se muestran en una lista filtrada dinámicamente con JavaScript. Voluntario HU01
RF02 Módulo de Gestión de Perfil Organizacional y Publicación de Actividades Se requiere un formulario HTML para que las Organizaciones creen su perfil y otro formulario para publicar actividades con campos básicos (título, ubicación, fecha, cupos). Los datos se guardan en tablas de base de datos relacionales. Organización HU02
RF03 Sistema de Inscripción Simple Implementación de un botón "Inscribirme" que al hacer clic registre la relación voluntario-actividad, reste 1 al cupo disponible, y muestre un mensaje de confirmación con alert() o un div de éxito. Voluntario HU03
RF04 Generador de Reportes Básicos con Consultas SQL El sistema debe ejecutar consultas SQL simples (COUNT, SUM) para calcular métricas (total de voluntarios, suma de horas, actividades completadas). Los resultados se muestran en HTML con opción de imprimir la página como PDF. Organización HU04, HU12
RF05 Vista de Lista de Eventos Inscritos Debe implementarse una página que haga una consulta SQL JOIN entre las tablas de voluntarios, inscripciones y actividades para mostrar todas las actividades en las que el voluntario está inscrito. Voluntario HU05
RF06 Módulo de Aprobación Manual de Solicitudes Se requiere una página para la Organización que liste todas las solicitudes con estado "pendiente", y botones "Aceptar" o "Rechazar" que ejecuten un UPDATE cambiando el estado. Organización HU06
RF07 Sistema de Notificaciones por Email Básico Implementación de envío de emails simples usando una librería que envíe un correo automático al voluntario cuando: se inscriba, haya un cambio en la actividad, o se publique una nueva actividad en su zona. Voluntario HU07
RF08 Herramienta de Envío de Mensajes Grupal Simple El sistema debe permitir a las Organizaciones seleccionar un grupo de voluntarios (por ejemplo, todos los inscritos a una actividad), escribir un mensaje, y enviar un email a cada uno usando un loop simple. Organización HU08
RF09 Sistema de Comentarios Simple Se requiere un formulario donde los voluntarios escriban un comentario, seleccionen una calificación de 1-5 estrellas, y opcionalmente suban una imagen. Los datos se guardan y se muestran en la página de detalle de cada actividad. Voluntario HU09
RF10 Módulo de Registro Simplificado con Validación Básica El sistema debe ofrecer un formulario de registro (nombre, email, contraseña, tipo de usuario) con validación HTML5 básica, y una página de bienvenida simple que explique cómo usar la plataforma. Usuario HU10
RF11 Sistema de Avisos por Email de Nuevas Actividades El voluntario puede marcar intereses. Cuando una organización publica una actividad, el sistema busca voluntarios con intereses coincidentes y les envía un email automático. Voluntario HU11
RF12 Página de Estadísticas Básicas para Organizaciones La plataforma debe proporcionar una página que ejecute consultas SQL agregadas para mostrar: total de horas de voluntariado, número de inscritos vs asistentes, y promedio de calificación. Organización HU12
RF13 Perfil de Voluntario con Lista de Actividades Completadas Se requiere una página de perfil que haga una consulta SQL JOIN para listar todas las actividades en las que el voluntario participó (estado='completado'), mostrando detalles y horas contribuidas. Voluntario HU13
RF14 Función de Pausa Simple de Perfil El sistema debe tener un botón en el perfil de la organización que cambie un campo booleano "activo" de true a false. El código debe ocultar las actividades de esa organización en las búsquedas. Organización HU14
RF15 Sistema de Mensajes Directos Básico Implementación de un formulario simple donde el voluntario escribe un mensaje dirigido a la organización de una actividad específica. El mensaje se guarda en una tabla "mensajes" con campos de vinculación. Voluntario HU15
