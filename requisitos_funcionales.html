REQUISITOS FUNCIONALES

ID Nombre del Requisito Descripción (Técnica) Usuario Historias de Usuario Relacionadas
RF01 Módulo de Registro de Usuarios El sistema debe implementar la función de alta de usuario (Sign Up), capturando los datos mínimos necesarios para diferenciar roles. Voluntario / Organización HU01
RF02 Validación de Datos de Perfil Se requiere una función de validación de campos obligatorios en el perfil para garantizar la calidad y completitud de los datos post-registro. Voluntario / Organización HU02
RF03 Módulo de Autenticación Segura Implementación de un proceso de Inicio/Cierre de Sesión (Log In/Out), utilizando protocolos seguros (JWT/OAuth) para validar credenciales. Voluntario / Organización HU03
RF04 Generación de Reportes Automáticos El sistema debe procesar datos de las actividades finalizadas y exponer una interfaz para la descarga automática de informes resumidos (ej. PDF/CSV). Organización HU04
RF05 Función de Geolocalización y Búsqueda Debe permitir la consulta de actividades en tiempo real utilizando la ubicación actual del usuario como parámetro principal de búsqueda. Voluntario HU05
RF06 Módulo de Filtrado por Taxonomía Se requiere la capacidad de refinar los resultados de búsqueda mediante una estructura categórica (taxonomía) basada en los intereses de voluntariado. Voluntario HU06
RF07 Motor de Recomendación de Contenido Implementación de un algoritmo de matching que sugiera actividades basándose en el historial de navegación y los intereses previamente declarados. Voluntario HU07
RF08 Interfaz de Creación de Actividades Diseño de un formulario estructurado que permita a las Organizaciones ingresar todos los metadatos necesarios para definir una actividad presencial. Organización HU08
RF09 Panel de Control de Contenido (CRUD) Módulo para que la Organización pueda consultar, editar o eliminar (CRUD) las actividades que ha publicado en la plataforma. Organización HU09
RF10 Módulo de Guías y Asistencia (Onboarding) El sistema debe ofrecer un flujo de onboarding o tutoriales interactivos que expliquen el uso de las funciones principales de la plataforma. Voluntario HU10
RF11 Sistema de Notificaciones Transaccionales Se requiere el envío automatizado de un mensaje de confirmación (e-mail/in-app) tras la correcta inscripción del voluntario en un evento. Voluntario HU11
RF12 Historial de Participación del Usuario Debe visualizar un registro cronológico de todas las actividades en las que el voluntario se ha inscrito y/o ha participado. Voluntario HU12
RF13 Alertas y Sincronización de Calendario La plataforma debe ofrecer la opción de sincronizar los eventos inscritos con un calendario personal (iCal o Google Calendar). Voluntario HU13
RF14 Módulo de Carga Multimedia Implementación de una función para subir archivos (imágenes/videos) asociados a la finalización de una actividad. Voluntario HU14
RF15 Submódulo de Micro-Reportes Se requiere un campo de texto con límite de caracteres para que el voluntario pueda escribir una breve reseña de su experiencia. Voluntario HU15
